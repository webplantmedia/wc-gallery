== Installation ==

1. Uzip the `wc-gallery.zip` folder.
2. Upload the `wc-gallery` folder to your `/wp-content/plugins` directory.
3. In your WordPress dashboard, head over to the *Plugins* section.
4. Activate *WordPress Canvas Gallery*.
